## Flarum UI Tab Extension
Flarum extension. Add a bottom tab to flarum on mobile.

### Future goals
- Add click animation.
- Custom icons and quantities

### Requirement
  - Flarum v1.0+
  - Flagrow Direct Links v0.2.0+

### Installation / Update
```
composer require itnt/flarum-uitab
```

### Remove
```
composer remove itnt/flarum-uitab
```
